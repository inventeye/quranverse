Quranverse

Download this project as a .zip file

Easily embed a verse from Qur'aan

Here's how to easily embed a verse from the Qur'aan on any website.


<script chapter="2" verse="255" 
   src="http://inventeye.github.io/quranverse/js/quran.js">
</script>

Here it is as an image for comparison:

Another example:


<script chapter="37" verse="137" 
   src="http://inventeye.github.io/quranverse/js/quran.js">
</script>

An example with english translation included. Just add another parameter called trans like so:


<script chapter="37" verse="123" trans="#translation"
   src="http://inventeye.github.io/quranverse/js/quran.js">
</script>
<div id='translation'<>/div>

Another example, this time with multiple verses. Just set the count like so:


<script chapter="37" verse="123" count="5"
   src="http://inventeye.github.io/quranverse/js/quran.js">
</script>
